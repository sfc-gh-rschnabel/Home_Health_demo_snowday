# Lincare Claims Submission Guidelines

## Policy Number: CLM-001
## Department: Revenue Cycle Management
## Effective Date: January 1, 2026
## Last Reviewed: March 15, 2026

## Purpose
To establish standardized procedures for the submission of Durable Medical Equipment (DME) claims to ensure maximum first-pass acceptance rates, regulatory compliance, and timely reimbursement across all 700+ Lincare operating centers.

## Scope
This policy applies to all billing personnel, intake coordinators, and branch managers involved in claims submission for Medicare, Medicaid, and commercial payer claims.

## HCPCS Code Requirements

### Respiratory Equipment
- E0431: Portable Oxygen Concentrator - Requires valid CMN (CMS-484), qualifying ABG/oximetry results, physician signature
- E0433: Stationary Oxygen Concentrator - Same documentation as E0431, plus home assessment
- E0601: CPAP Device - Requires sleep study (PSG or HSAT) showing AHI >= 5, face-to-face evaluation within 90 days, compliance monitoring plan
- E0470: BiPAP Device (without backup rate) - Requires failed CPAP trial documentation, sleep study with AHI >= 5
- E0465: Home Ventilator - Requires hospital discharge documentation, pulmonologist certification, caregiver training verification
- E0570: Nebulizer - Requires prescription with specific medication, diagnosis supporting medical necessity

### Mobility Equipment
- K0001: Standard Wheelchair - Requires face-to-face examination, mobility assessment, home evaluation
- K0004: High-Strength Wheelchair - Additional justification for upgraded features, weight documentation
- E0143: Walker with Wheels - Requires gait assessment, PT evaluation recommended
- E0260: Hospital Bed (Semi-Electric) - Requires medical necessity documentation showing inability to reposition independently

## Modifier Requirements
- RR: Rental item - Required for capped rental equipment during rental period
- NU: New equipment purchase - Required when purchasing instead of renting
- KX: Medical necessity documentation on file - Critical for oxygen and CPAP claims
- GA: Waiver of liability on file (ABN) - Required when coverage uncertain
- GY: Statutorily non-covered item - Required for items outside Medicare benefit

## Clean Claim Standards
A clean claim must contain:
1. Valid patient demographics (name, DOB, address matching Medicare records)
2. Correct HCPCS code with appropriate modifiers
3. Valid ICD-10 diagnosis code supporting medical necessity
4. Referring physician NPI (must be enrolled in PECOS)
5. Place of service code (12=Home, 13=Assisted Living, 14=Group Home)
6. Date of service matching delivery/setup date
7. Certificate of Medical Necessity (if applicable) with all required fields completed
8. Prior authorization number (when required by payer)
9. Lincare NPI and taxonomy code

## Submission Timelines
- Medicare: Submit within 12 months of date of service (timely filing limit is 365 days)
- Medicaid: Varies by state, typically 90-180 days
- Commercial payers: Per contract terms, typically 90-120 days
- Target: Submit all claims within 5 business days of service/delivery date

## Common Rejection Prevention
1. Always verify patient eligibility before delivery/setup
2. Confirm referring physician NPI is active in PECOS
3. Ensure CMN is complete with no blank required fields
4. Verify equipment HCPCS code matches what was actually delivered
5. Check diagnosis code supports medical necessity for the specific equipment
6. Confirm prior authorization was obtained before service date (when required)
7. Double-check modifier usage matches rental/purchase/replacement status
